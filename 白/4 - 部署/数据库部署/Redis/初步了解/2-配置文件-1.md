# Redis 配置

Redis 的配置文件位于 Redis 安装目录下
文件名为 `redis.conf`(Windows 名为 redis.windows.conf)

可以通过 `CONFIG` 命令**查看或设置**配置项。

> 可自动补全

## 语法

Redis CONFIG 命令格式如下：

`redis 127.0.0.1:6379> CONFIG GET CONFIG_SETTING_NAME`

### 实例

```sh
127.0.0.1:6379> CONFIG GET loglevel

1) "loglevel"
2) "notice"
```
