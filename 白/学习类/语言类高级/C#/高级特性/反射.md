# 反射

## 获取对象的类型

    `Type type = Object.GetType()` = `typeof(Object)`
        实例 ：
        System.Console.WriteLine(intNum.GetType().ToString());

## 获取对象的参数 Property

    type.getProperties();
    type.getProperty();

## 获取对象的方法 Method

    type.getMethod();
    type.getMethods();

## 获取对象的成员 Member

    type.getMember();
    type.getMembers();
