## 逻辑运算符

> `and` `or` `not`

与 或 非

`and` 的俩边 只要有一个为 `False` ,则结果为 `False` , 其余都为 `True`

`or` 的俩边 只要有一个为 `True` ,则结果为 `True` , 其余都为 `False`

只有 0 为 `False`,其余都为`True`

非 等同于取反

    3. 重复输出 `*`<br>
        > str[:6] + str[6]*3 + str[7:]
    4. 截取字符串 `[:]`  左闭右开 ， 最后一位可以用 `-1`
        > str[-2:] str[0:2] str[:6]
    5. 字符串的值不可以直接修改 (所有更改操作，多数都能用这种)
        > str[2] = 'a'   错误
        > str = str[:2] + 'a' + str[3:]     正确
