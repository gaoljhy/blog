## 输入输出函数

1. `print()`
    ```
    print('hello world')

    demon = ("hello world","every one")
    print(demon)
    与
    print("hello world","every one")

    demon = ("hello world","every one")
    print(demon,"now")


    ```

2. `input()`

    读取一个对象
    ```
    str = input()
    print(str)

    str = input("please input:")
    print(str)
    ```

3. 数值型函数

    |函数|意义|
    |----|:---:|
    |`int()`| 转换为整数型
    |`bool()`|转换为bool|
    |`float()`|转换为浮点型|
    |`complex()`|转换成复数|

注意： 

>`“”`,`None`,0 转 bool 为 `False`
